# Работаем с ветками
## Создаем новую ветку командой:
    `git branch branchname`
## Переключиться в новую ветку: 
    `git checkout surname`
## Подготовить новый файл для комита: 
    `git add` 
## Проверить статус репозитория: 
    `git status`
## Сделать комит: 
    `git commit -m “...”`
## Отправить изменения на сервер: 
    `git push`
